Payroll for Construction & Contracting Industry Odoo Apps

Project Job Costing Construction Payroll Odoo Apps helps to generate payslip for contractor payroll system, this odoo apps added different salary rules for work package decided for contractor and workers for job costing industry which helps to generate payroll payslip for Project Job Costing contractors. It also provide different reports for timecard and payslip payroll.

This Odoo app works for Enterprise Edition/Odoo.sh as Payroll is moved to Enterprise Edition in Version 13.

Features

Timesheet/Timecard Creation

Allow you to created Timesheet/Timecard with work type.

Configure Work Type

Allow to configure various work types.

Salary Rules with Different Work Type

Create salary rules according to the work type.

Timesheet/Timecard PDF Report

Timesheet/Timecard pdf report generation.

Payslip with Detail Information of Timecard Lines and Work Types

Payslip contains the detail information of timesheet and PAID on confirmation.

Employee Payslip PDF Report

Employee payslip report contain detail information on work types, and its timesheet-lines.
